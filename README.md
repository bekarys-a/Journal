# Journal

```
USAGE:
    journal [OPTIONS] <COMMAND>

OPTIONS:
    -h, --help       Prints help information
    -v, --version    Prints version information

COMMANDS:
    view <PATH>             прсмотр
    add <PATH> <DATE>       добавить дату в файл
    delete <PATH> <DATE>    удалить дату из файла
```
